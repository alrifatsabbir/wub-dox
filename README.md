WUB DOX is a specialized tool designed for students to generate professional university cover pages (Assignment, Lab Report, Forum, etc.) in seconds. It focuses on simplicity, speed, and privacy.

## âœ¨ Features

- **Quick Generation:** Create high-quality cover pages in less than a minute.
- **Multiple Templates:** Supports Assignments, Lab Reports, Forums, and Homework.
- **Dual Language:** Full support for both **English** and **Bengali**.
- **Privacy Focused:** No data is stored on any server. Everything is processed in your browser.
- **Interactive Tour:** Built-in guide for first-time users using `driver.js`.
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop.

## ğŸ› ï¸ Tech Stack

- **Frontend:** React.js (Vite)
- **Styling:** Tailwind CSS, Framer Motion
- **Icons:** Lucide React
- **Translation:** i18next
- **Animations:** Framer Motion
- **Tour:** Driver.js

## ğŸ”’ Privacy & Data

- All user input is stored in the browser's `localStorage`.
- PDF generation happens purely on the **Client-side**.
- Your data never leaves your device.
